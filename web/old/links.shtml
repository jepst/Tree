Syntax Tree Editor isn't the only program that you can use to create syntax tree diagrams. Here are some other products that may meet your needs.

  • TreeForm is a full-featured, graphical syntax tree diagram editor. It's written in Java, and so should run on just about any operating system.
  • Tree Builder is another Java-based solution that also features built-in grammar checking.
  • Linguistic Tree Constructor is a graphical tool to interactively generate syntax tree diagrams.
  • Trees is a another graphical, interactive tool, with particularly strong linguistic features.
  • Arboreal is not a program, but a font that contains syntax tree diagram components as characters.
  • Qtree is a package for LaTeX.
  • phpSyntaxTree is a web-based solution that generates syntax tree diagrams based on bracketed textual input.
  • Syntax Tree Drawer is a tool to generate SVG files for syntax tree diagrams.
  • Lehner's Prolog Tree Drawing Program generates syntax tree diagrams from Prolog input.
